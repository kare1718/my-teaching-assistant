# 나만의 조교 PLAYBOOK

> 상용화 리빌드 참고 카드. 마누스 15개 전략 문서를 요약·재구성한 실행 기준.
> 최종 개정: 2026-04-15 · 현황: DB 65+ 테이블 / API 160+ / React+Vite+Express+Supabase

---

## 1. 제품 정의

**한 줄 정의**
> 나만의 조교는 **학원 운영 자동화와 재원 유지 강화를 동시에 돕는 올인원 SaaS**다. "학원관리 프로그램"이 아니다.

**3층 구조 (Core / Pro / First Class)**

| 계층 | 의미 | 주요 기능 |
|---|---|---|
| **Core** | 매일 반드시 쓰는 운영 기본 | 출결, 학생/반 관리, 수납, 공지, 상담, 보호자 확인, 메시지 발송 |
| **Pro** | 운영 효율·통제력 확장 | 자동화 엔진, 상담 CRM, 운영 리포트, 메시지 고급 운영 |
| **First Class** | 차별화·리텐션 강화 | 게이미피케이션(XP/퀘스트/아바타/상점), AI 업무보조, 고급 리포트, 브랜딩 |

원장이 돈을 내는 이유 = 미납 감소, 결석 대응, 상담 효율, 재원 유지. 게이미피케이션·AI는 이 가치에 연결될 때만 의미 있음.

## 2. 핵심 원칙 10개

1. **운영 코어 선완성** — Core가 잠기기 전에 AI·게이미피케이션을 전면에 두지 않는다.
2. **먼저 잠그고, 정리하고, 확장한다** — 상용화 기반 → 코어 UX → 프리미엄 순.
3. **관리자 홈은 "오늘 할 일"** — KPI 대시보드 아님.
4. **학생 상세 = 운영 허브** — 출결/수납/상담/소통이 학생 단위로 연결.
5. **원장 언어로 설명** — "퀘스트/XP" 아닌 "미납 감소/상담 효율"로 판다.
6. **역할 분리** — 관리자는 업무, 보호자는 안심, 학생은 참여 중심.
7. **요금제는 운영 성숙도 기준** — 학생 수 기준 아님.
8. **예외 처리가 품질을 결정** — 수납 예외가 안 되면 엑셀로 복귀한다.
9. **게이미피케이션은 First Class 전용·옵션형** — 학원 유형별 ON/OFF.
10. **투명성** — 메시지 비용, 해지/환불, 사업자 정보는 숨기지 않는다.

## 3. 요금제 구조 (확정)

> ⚠️ **모든 가격은 월 요금 / VAT 별도**. UI/랜딩/영수증/가격표 어디든 반드시 "VAT 별도" 문구 병기.
> 신규 가입 시 **30일 무료 체험** (Pro 기능 체험, `trial_ends_at` 기반). 만료 후 Free로 자동 전환. 별도 Trial 플랜 없이 `subscriptions.status='trial'`로 관리.

| 플랜 | 월 가격 (VAT 별도) | 학생 수 | 타겟 | 핵심 가치 | 포함 범위 |
|---|---|---|---|---|---|
| **Free** | 0원 | **15명** | 1인 과외, 도입 체험 | 리드 확보 | 성적/출결/공지/자료/Q&A만. 자동화·CRM·게이미피케이션·브랜딩 전부 제외 |
| **Starter** | **49,000원** | **50명** | 소형 학원 (엑셀 탈출) | 운영 기본팩 | + 학생 관리·수납 기본·SMS·보호자 앱·리뷰·기본 상담 (Core 전체) |
| **Pro** | **129,000원** | **100명** | 성장 학원 | 시간 절감 / 운영 자동화 | + 자동화 엔진, 상담 CRM, 고급 리포트, 메시징 정책, 리드 파이프라인, 수납 예외, AI 리포트 |
| **First Class** | **별도 문의** | 무제한/협의 | 관리형·입시형·브랜딩형 | 리텐션·차별화 | + 게이미피케이션, 랭킹/상점/칭호, 어휘·지식·독해·OX 퀴즈, 아바타, AI 문제 생성, 포트폴리오, 브랜딩, 명예의 전당 |

**키(subscription_tier DB 값)**: `free` / `starter` / `pro` / `first_class`
**레거시 호환 매핑**: `trial→free`, `basic→starter`, `standard|growth→pro`, `premium→first_class`

**연간 결제 (15% 할인, VAT 별도)**
- Starter: 월환산 41,650원 (연 499,800원)
- Pro: 월환산 109,650원 (연 1,315,800원)
- First Class: 연간 제공 없음 (별도 문의)

**원칙**
- Free는 도입 체험용 (유료 전환 이유 반드시 남김)
- Starter는 "엑셀 탈출" 한 문장으로 판매
- Pro는 "직원이 적어도 돌아간다"를 판매
- First Class는 "재원이 더 오래 다닌다"를 판매
- **메시지(SMS/알림톡) 비용은 구독료와 분리 투명 표시**
- **모든 가격 표기 옆에 반드시 "VAT 별도" 병기**

## 6. 관리자 홈 구성 원칙

> 홈은 "데이터가 많아 보이는 화면"이 아니라 **원장이 30초 안에 일하기 시작하는 화면**이다.

| 영역 | 내용 | 목적 |
|---|---|---|
| **오늘 처리할 일** (최상단) | 미납 학생, 결석 학생, 상담 예정, 승인 대기, 발송 실패 | 즉시 행동 |
| **빠른 실행** | 출결 등록, 청구 생성, 공지 발송, 상담 메모 | 클릭 단축 |
| **위험 알림** | 7일 결석 증가, 장기 미납, 이탈 위험 학생 | 선제 대응 |
| **핵심 지표** (보조) | 재원 수, 출결률, 미납률, 상담 완료율 | 상태 파악 |
| **최근 활동** | 누가 무엇을 했는지 | 운영 가시성 |

KPI 카드는 두 번째 레이어. 행동 카드가 항상 위.

## 7. UI/UX 디자인 원칙

**키워드**: Clean · Calm · Efficient · Trustworthy · Expandable

**디자인 토큰**
- Primary: `#102044` (딥 네이비, 신뢰·운영)
- Accent: `#004bf0` (강조, CTA)
- 폰트: **Paperlogy** (Korean), 시스템 fallback
- Success: 정돈된 그린 / Warning: 소프트 오렌지 / Danger: 채도 낮은 레드

**레이아웃**
- 관리자: 좌측 네비 고정 + 상단 바 + 메인 + 보조 패널
- 보호자/학생: 모바일 우선, 하단 탭바 4~5개

**컴포넌트 원칙**
- 상태는 **색 + 텍스트** 병행 (색만으로 의미 전달 금지)
- 테이블: 첫 열 고정, 필터 상단 고정, 일괄 작업 지원, 행 클릭 시 드로어
- 폼: 한 화면 한 목적, 실시간 검증, 기본값 제공, 저장 후 다음 행동 제안
- Empty state: 다음 행동 유도 문구 포함
- 마이크로카피: "에러 발생" ✗ → "일부 메시지가 발송되지 않았습니다. 실패 대상만 다시 보낼 수 있습니다" ✓

**상태 표준어 (제품 전체 통일)**
- 출결: 출석/지각/결석/조퇴/보강
- 수납: 청구 예정/청구 완료/부분 납부/납부 완료/미납/환불
- 메시지: 임시/예약/발송 중/발송 완료/일부 실패/실패
- 상담: 예정/완료/후속 필요/보류

## 14. 절대 원칙 (출시 전 금지 사항)

> 아래는 출시 전까지 **절대 하지 말 것**. 하나라도 어기면 출시 자격 상실.

1. **운영 코어 미완성 상태에서 게이미피케이션·AI를 전면에 내세우지 않는다.**
2. **멀티테넌시 자동 테스트 없이 유료 고객을 받지 않는다.**
3. **수납 예외 8가지 중 하나라도 막혀 있으면 Starter 이상 판매하지 않는다.**
4. **관리자 홈을 KPI 대시보드로 두지 않는다.** ("오늘 할 일" 중심 고정)
5. **Free 플랜에 자동화·고급 CRM·AI·수납을 넣지 않는다.** (유료 전환 이유 소실)
6. **게이미피케이션을 Starter/Pro 기본값으로 켜두지 않는다.** (First Class 전용, 기본 OFF)
7. **메시지 비용을 구독료에 감춰 합치지 않는다.** (단가·이력·충전 투명 표시)
14. **가격을 "VAT 포함"으로 표기하거나 VAT 별도 문구를 누락하지 않는다.**
15. **별도의 Trial(체험) 티어를 부활시키지 않는다.** (체험은 subscription status='trial' + trial_ends_at=30일 후, 만료 시 Free로 전환)
8. **푸터 법적 고지 (상호·대표자·사업자번호·주소·전화·이메일·약관·개인정보처리방침) 없이 결제 화면을 노출하지 않는다.**
9. **감사로그 없이 삭제·환불·권한 변경 기능을 제공하지 않는다.**
10. **백업·복구 리허설 없이 고객 데이터를 받지 않는다.**
11. **"학원관리 프로그램"이라는 표현을 제품·랜딩·마케팅 어디에도 쓰지 않는다.** (포지셔닝 파괴)
12. **"나만의 조교"(SaaS)와 특정 학원 브랜드 네이밍("서강인 국어" 등)을 혼용하지 않는다.**
13. **신규 기능 추가는 W11 이후 금지.** (QA·안정화만)
